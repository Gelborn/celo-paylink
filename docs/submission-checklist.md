# Submission Checklist

- Deploy `PayLinkProfile.sol` on Celo mainnet.
- Verify the contract on an explorer.
- Set `NEXT_PUBLIC_CONTRACT_ADDRESS_MAINNET` in the production env.
- Create the Talent project and add the payout wallet.
- Add the public GitHub repo and contract address to the project.
- Record at least a few real stablecoin payments before the leaderboard cutoff.
- Test the public pay link inside MiniPay on phone.
- Capture screenshots for:
  - Landing page
  - `/my` profile editor
  - `/u/[handle]` payment card
  - Success screen
- Upload a demo video under 4 minutes.
- Upload a short deck under 10 slides.
